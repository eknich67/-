Данный код предостовляет пользователю задачи на вычисление индикта и перевод дат из разных каледарных стилей (мартовский, ультрамартовский, сентябрьский) в современный. Также присутствует опция в любой момент запросить справочный материал с алгоритмом опрдеделения индикта и перевода дат. Код находится в файде "Задачи_на_перевод.py"
